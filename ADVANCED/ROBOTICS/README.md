# ADVANCED/ROBOTICS — C for Robotics Concepts

All programs simulate robotics concepts in software — no hardware required.

| File | What it demonstrates |
|------|----------------------|
| `pid_controller.c` | Discrete PID control loop, step-response simulation, CSV output |
| `kinematics_2d_arm.c` | Forward + inverse kinematics for a 2-DOF planar arm |
| `pathfinding_grid_c.c` | A* search on a 2D grid with ASCII visualization |
| `sensor_fusion_simple.c` | Complementary filter merging two noisy simulated sensors |
| `realtime_scheduler_sim.c` | Cooperative RTOS-style periodic task scheduler |

## Build all
```
.\build.bat
```
