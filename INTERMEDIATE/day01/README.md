# Intermediate Day 01 — Data Structures I: Linked Lists

## Objectives
- Implement singly and doubly linked lists from scratch
- Insert, delete, and traverse nodes
- Understand heap allocation for nodes
- Compare linked lists vs arrays (tradeoffs)

## Concepts Covered
- Node struct: `struct Node { int data; struct Node *next; }`
- Head pointer, tail pointer
- Insert at head, tail, position
- Delete by value
- Doubly linked: prev + next pointers
