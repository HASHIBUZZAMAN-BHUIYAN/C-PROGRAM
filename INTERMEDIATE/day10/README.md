# Intermediate Day 10 — Capstone: CLI Hash-Table Task Manager

## Objectives
Apply the Intermediate-track concepts in one real CLI tool:
- Hash table for O(1) task lookup by ID
- Doubly linked list for ordered display
- File persistence (load/save CSV)
- Priority queue concept (sort by priority)

## Features
- Add/delete/list tasks with name, priority (1-5), and status
- Search by ID (hash table, O(1)) or keyword (linear scan)
- Persist to disk as `tasks.csv`

## Time Estimate
~3.5 hours

## How to Build & Run
```
.\build.bat
capstone.exe
```
