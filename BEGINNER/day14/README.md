# Day 14 — Capstone: Student Record Manager

## Objectives
Apply everything from Days 1-13 in one cohesive program:
- Structs + dynamic arrays (malloc/realloc/free)
- File I/O (save/load a CSV database)
- Functions, pointers, string handling
- User menu loop with conditionals and loops

## Features
- Add student records (name, ID, age, GPA)
- List all students
- Search by name or ID
- Delete a record
- Sort by GPA
- Save to / load from a file automatically on exit/start

## Time Estimate
~3 hours

## How to Build & Run
```
.\build.bat
capstone.exe
```
