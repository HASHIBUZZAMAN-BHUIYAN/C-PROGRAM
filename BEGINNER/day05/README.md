# Day 05 — Functions: Declarations, Definitions, Parameters, Return Values

## Objectives
- Declare and define your own functions
- Pass arguments by value
- Return values from functions
- Understand function prototypes (forward declarations)
- Write void functions and functions with multiple parameters

## Concepts Covered
- `return_type function_name(param_type param) { ... }`
- Forward declarations (prototypes above main)
- Pass-by-value: functions get copies of arguments
- Recursion preview (factorial)
