# Day 01 — C Basics: Hello World, Variables, Data Types

## Objectives
- Write and compile your first C program
- Understand the structure of a C file (includes, main, return)
- Use `printf` to display output and `scanf` to read input
- Declare and use int, float, double, char variables
- Understand basic data type sizes

## Concepts Covered
- `#include <stdio.h>`, `main()`, `return 0`
- `printf` format specifiers: %d %f %c %s %lf
- `scanf` for user input
- Variable declaration and initialization
- `sizeof` operator
