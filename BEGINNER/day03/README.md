# Day 03 — Conditionals: if/else and switch

## Objectives
- Write if, if/else, and if/else-if chains
- Use the switch statement for multi-way branching
- Understand the ternary operator
- Understand truthiness in C (0 = false, nonzero = true)

## Concepts Covered
- `if (condition) { } else { }`
- `else if` chains
- `switch (expr) { case X: ... break; default: }`
- Ternary: `condition ? value_if_true : value_if_false`
- Dangling-else pitfall
