# Day 02 — Operators, Expressions, Type Casting, const

## Objectives
- Use arithmetic, relational, logical, and bitwise operators
- Understand operator precedence
- Perform type casting (implicit and explicit)
- Use `const` for read-only variables

## Concepts Covered
- Arithmetic: + - * / %  Increment/decrement: ++ --
- Relational: == != < > <= >=
- Logical: && || !
- Assignment shortcuts: += -= *= /= %=
- Type casting: (int)3.9 = 3
- `const int MAX = 100;`
