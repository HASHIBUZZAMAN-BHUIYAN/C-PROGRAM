# Day 04 — Loops: for, while, do-while, break, continue

## Objectives
- Write for, while, and do-while loops
- Use break to exit a loop early
- Use continue to skip an iteration
- Write nested loops
- Understand infinite loops and how to avoid them

## Concepts Covered
- `for (init; condition; update) { }`
- `while (condition) { }`
- `do { } while (condition);`  — always runs at least once
- `break` and `continue`
- Nested loops and the multiplication table
