# Day 09 — Pointers II: Pointers to Pointers, Function Pointers

## Objectives
- Understand double pointers (pointer-to-pointer)
- Use double pointers for "output" parameters (e.g., allocating memory in a function)
- Declare and call function pointers
- Use function pointers to build a simple dispatch table

## Concepts Covered
- `int **pp` — pointer to pointer
- `int (*fp)(int, int)` — function pointer
- Function pointer arrays (dispatch table)
- `const` with pointers: `const int *p` vs `int * const p`
