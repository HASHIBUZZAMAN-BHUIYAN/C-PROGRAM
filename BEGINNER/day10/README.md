# Day 10 Ś Structs and Unions

## Objectives
- Define and use structs to group related data
- Access struct members with . and ->
- Nest structs and create arrays of structs
- Understand unions (same memory, different types)
- Use typedef for cleaner struct names

## Concepts Covered
- `struct Point { int x; int y; };`
- `typedef struct { ... } TypeName;`
- Passing structs to functions (by value and by pointer)
- `struct Student students[30];`
- `union Data { int i; float f; char c; };`
