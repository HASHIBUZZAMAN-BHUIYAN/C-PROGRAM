# Day 06 — Arrays (1D and 2D)

## Objectives
- Declare, initialize, and access 1D arrays
- Iterate arrays with loops
- Pass arrays to functions
- Work with 2D arrays (matrices)
- Understand that arrays are zero-indexed

## Concepts Covered
- `int arr[5] = {1, 2, 3, 4, 5};`
- Array indexing: `arr[0]` through `arr[n-1]`
- Passing arrays: `void func(int arr[], int size)`
- 2D arrays: `int matrix[3][3]`
