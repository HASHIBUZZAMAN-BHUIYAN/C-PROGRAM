# Day 12 — File I/O

## Objectives
- Open and close files with fopen/fclose
- Read and write text files with fprintf/fscanf/fgets
- Read and write binary files with fread/fwrite
- Handle file errors gracefully
- Use fseek/ftell for random access

## Concepts Covered
- `FILE *f = fopen("file.txt", "r"/"w"/"a"/"rb"/"wb");`
- `fprintf`, `fscanf`, `fgets`, `fputs`
- `fread`, `fwrite` for binary data
- `fseek(f, offset, SEEK_SET/SEEK_CUR/SEEK_END)`
- `ftell`, `feof`, `ferror`
